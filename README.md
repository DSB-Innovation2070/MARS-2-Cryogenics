MARS: Magnetic Active Refrigeration Systems
High-Frequency Solid-State Cryogenics
Project Overview

This repository contains the core logic and technical framework for the MARS project, focusing on helium-free, vibration-less cooling powered by the Boghian-Shift high-frequency gating algorithm (1 GHz).
